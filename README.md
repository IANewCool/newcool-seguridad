# NewCool Seguridad - Seguridad Ciudadana Chile

Portal educativo sobre seguridad ciudadana y servicios policiales en Chile.

## Demo

[https://seguridad.newcool.io](https://seguridad.newcool.io)

## Caracteristicas

- Buscador de comisarias
- Numeros de emergencia
- Guia de tipos de delitos
- Calculadora de prescripcion
- Derechos del detenido

## Tecnologias

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo

```bash
npm install
npm run dev
```

## Parte del Ecosistema NewCool

Este modulo es parte de [NewCooltura Informada](https://informada.newcool.io) - Portal de instituciones publicas de Chile.
